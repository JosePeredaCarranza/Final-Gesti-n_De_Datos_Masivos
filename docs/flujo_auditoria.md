# Flujo de Auditoría

La auditoría registra el ciclo de vida del dato desde la descarga hasta las capas analíticas.

## Campos mínimos

- pipeline_id
- pipeline_name
- pipeline_version
- started_at
- finished_at
- file_name
- source_url
- local_path
- status
- error_message
- size_bytes
- sha256
- records_processed
- execution_time
